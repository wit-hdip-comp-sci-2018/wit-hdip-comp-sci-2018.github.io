Semester 1
Programming Fundamentals

Provide an introduction Computer Programming using the Java language. The course assumes no prior programming experience, and covers the foundation of programming and key topics in the Java Language.

Web Development

Uncover the basics components a web page, learn the fundamental of structure of the HyperText Markup Language (HTML), the role Cascading Style Sheets (CSS). Transition from exploring the construction of Web Sites to simple Web Applications in Java.
